# Тестирование интернет-магазина

## **Описание**
В этом проекте на начальном этапе была проведена работа по классификации требований заказчика, далее на основе проанализированных требований был разработан набор тест-кейсов и сделано тестирование, в ходе которого были обнаружены баги и составлены на них баг репорты. После исправления багов был проведен финальный этап регрессионного тестирования.
